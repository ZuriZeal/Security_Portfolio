# Fail2Ban IPS Evidence (AWS EC2)

This folder contains screenshots proving a working Fail2Ban Intrusion Prevention System (IPS) on AWS EC2:
- Active jails (apache-auth, apache-badbots, apache-recon)
- Permanent ban confirmation (bantime = -1)
- firewalld rich rules enforcing http/https blocks
- fail2ban.log ban events
- Service running status
